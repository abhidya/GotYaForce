|---|---|
| A | Confirm / context action |
| B | Hold-charge shot / cancel |
| X | Special |
| Y | Borg switch shortcut |
| L / R | Camera rotate / lock |
| Z | Buster / force command |
| D-Pad | Borg select during battle |
