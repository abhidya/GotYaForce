# Function Explorer

This is a live browser of all 11,972 decompiled functions from the Ghidra
full-corpus export. Filter by name, address, or topic; click any row to read
the C source inline and inspect caller/callee counts. Original source files
live in `research/decomp/ghidra-export/` and are served on-demand from
`public/chunks/`.

For the checked-in Ghidra project, OGhidra analysis sessions, export inventory,
and corpus-level metrics, see the [Ghidra Corpus](./ghidra-corpus).
