# Story / Adventure mode

5 chapters, ~52 missions, recruit flow, plot beats.

**Source:** `research/game-design/adventure-mode.md`

<StatusBadge status="likely" />

## Chapter structure

Each chapter has a fixed set of missions with branching recruit unlocks. The
story mode dispatcher is part of the 11-mode top-level menu (see
`research/decomp/index/title-main-menu-flow.md`).

## Recruits

Borg unlocks are gated by chapter progression and feed into the Gotcha-Box
drop pool. See [Force → Gotcha-Box](../force/gotcha-box) for the spawn pool
table.
